# Rustic Palm Listing Creator

This is the deployment-ready version of the Etsy digital wall art tool.

## Deploy on Vercel

1. Unzip this folder.
2. Upload the whole folder to a new GitHub repository.
3. Go to Vercel.com.
4. Select New Project.
5. Import the GitHub repository.
6. Click Deploy.

Vercel will detect Vite automatically.

## Local testing, optional

Install Node.js, then run:

npm install
npm run dev
